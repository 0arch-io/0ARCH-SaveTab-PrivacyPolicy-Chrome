# SaveTab - Chrome Web Store Listing

## Short Description (132 characters max)
Save and restore tab collections with one click. Organize workspaces, suspend tabs, and use AI-powered smart grouping.

## Detailed Description

**SaveTab** is a powerful yet elegant tab management extension that helps you organize, save, and restore your browsing sessions with ease.

### Key Features

**Tab Collections**
- Save all your current tabs as a named collection with one click
- Restore entire collections instantly
- Close tabs when saving to reduce memory usage
- Beautiful visual organization with custom colors and icons

**Smart Workspaces**
- Create dedicated workspaces for different projects or contexts
- Switch between work, personal, research, and custom workspaces instantly
- Each workspace maintains its own set of collections

**AI-Powered Smart Grouping**
- Use Google Gemini AI to automatically organize tabs by topic
- Intelligent categorization of your browsing sessions
- Create meaningful groups from chaotic tab collections

**Tab Suspender**
- Automatically suspend inactive tabs to save memory
- Customizable suspension delay (5 minutes to 2 hours)
- Beautiful suspension page with quick wake-up
- Can save up to 100MB+ per suspended tab

**Beautiful Dark Interface**
- Ultra-modern, clean design inspired by premium applications
- Immersive dark theme that's easy on the eyes
- Side panel integration for seamless workflow
- Customizable colors and themes

**Full Control**
- Export/import collections as JSON backups
- Sort collections by date, name, or tab count
- Search and filter through your collections
- Complete privacy - all data stored locally

### Perfect For

- Researchers managing multiple topics
- Developers working on different projects
- Students organizing study materials
- Anyone with too many tabs open

### Privacy First

SaveTab respects your privacy:
- All data stored locally on your device
- No tracking, analytics, or data collection
- No external servers (except optional AI features)
- Open source and transparent

### How to Use

1. Click the SaveTab icon or press Ctrl+Shift+S
2. Save your current tabs as a collection
3. Close tabs to free up memory
4. Restore collections anytime with one click
5. Use workspaces to organize different contexts

### Optional AI Features

Enable AI Smart Grouping with your own free Google Gemini API key to automatically categorize and organize your tabs intelligently.

### Requirements

- Chrome version 89 or higher (for tab groups support)
- Optional: Google Gemini API key for AI features
